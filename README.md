# browser-os

> A module to inform the Browser Operating System

## How?

_Start:_

1. Install with `npm i --save browser-os`
2. Import and use:
````javascript
var OS = require('browser-os')()

//or
console.log(require('browser-os')())

//or
var OS = require('browser-os')
console.log(OS())

/*
* Return STRING with OS
* Ex: Windows 10
*/
````
